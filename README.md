# AeroclubDeiMarsi - Fly School Management System

A comprehensive web application for managing a fly school, including member management, aircraft rental tracking, and payment processing.

## Features

- **Member Management**: Register and manage school members with profiles
- **Aircraft Management**: Manage aircraft inventory and availability
- **Rental System**: Book and track aircraft rentals
- **Payment Processing**: Record and track rental payments
- **Dashboard**: Overview of school operations and statistics
- **Reporting**: Generate reports on rentals, payments, and member activity

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (with frameworks)
- **Backend**: Node.js/Express or Python/Flask
- **Database**: PostgreSQL or MongoDB
- **Authentication**: JWT-based user authentication
